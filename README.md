# Agentic AI Example

A minimal example of an agentic AI architecture (conversation, tools, agent state). This repository contains small modules used for experimenting with agent design patterns.

# Ref:
https://medium.com/@iamanraghuvanshi/building-an-ai-agent-from-scratch-with-pure-python-in-2026-229af2a9b8ce


## Quick Start

- Create and activate the virtual environment (Windows PowerShell):

```powershell
python -m venv venv
& .\venv\Scripts\Activate.ps1
```

- Run the example runner:

```powershell
py -m runner.py
```

## Project Structure

- `AgentClass.py` — dataclasses for messages, tools, actions, and agent state.
- `LLMClient.py` — LLM wrapper used by agents.
- `ReActAgent.py` — example agent implementation.
- `runner.py` — small entrypoint to exercise the agent.
- `tool.py` — helper tool implementations.
